Modul 1

🧠 AI Pravni Agent

Srce platforme. Advokat postavlja pitanje — AI agent pretražuje zakone, sudsku praksu i relevantna ministarstvena mišljenja i daje konkretan odgovor sa citatima.

  • ⚡
    Odgovor za 3–5 sekundi
    GPT-4o + Pinecone vektorska pretraga zakona
  • 📖
    Citira srpski zakon
    ZPP, KZ, ZUP, Zakon o radu, ZPD, Porodični zakon i više
  • ⚖️
    Sudska praksa
    Presude Vrhovnog i Apelacionog suda (ratio decidendi)
  • 🏛️
    Ministarstvena mišljenja
    Filtrirana po relevantnosti — threshold 0.62 (anti-hallucination)
  • 💬
    Istorija pitanja
    Svaki odgovor sačuvan u predmetu, pretraživ

Modul 4

💰 Naplata, Tajmer i AKS Tarifa

Automatski obračun po Advokatskoj komori Srbije (Sl. gl. 56/2025). Nema više izgubljenih sati.

⏱️

Live tajmer

Start/stop jednim klikom. Automatski kreira naplatnu stavku po zaustavljanju. Minimalna satnica 7.500 RSD/h.

AKS Tarifa

🔄

Recurring fakture

Modul 7 · Novo

🕸 Knowledge Graph

Interaktivna mapa odnosa predmeta. Klikni na "Graf" tab — AI crta veze između predmeta, klijenata, dokumenata, rokova i zakona.

5 tipova čvorova

Predmet — centralni čvor (veći radijus)
Klijenti / stranke — tužilac, tuženi
Dokumenti — sa tipom dokaza
Rokovi — sa datumom isteka
Zakoni / akteri — pravni osnov, sud

Kako funkcioniše

  • AI povlači sve veze iz baze podataka predmeta
  • Force-directed algoritam raspoređuje čvorove vizuelno
  • Hover na čvor → tooltip sa detaljima
  • Strelice pokazuju smer veze (predmet → dokument)
  • Jake veze (pravni osnov, stranke) su deblje
💡 Savršeno za vizuelno objašnjavanje predmeta klijentu na sastanku

Bezbednost i usaglašenost

🛡️ Bulletproof sigurnost

Advokatska tajna nije kompromis. Svaki sloj sistema je dizajniran da zaštiti podatke klijenata.

Oblast Implementacija Standard
🔐 Autentifikacija JWT token + triple verifikacija (signature, expiry, user_id) OAuth2
🗄️ Izolacija podataka Row Level Security (RLS) — korisnik vidi SAMO svoje predmete PostgreSQL RLS
🔒 Osetljivi podaci JMBG, PIB, pasoš → AES-256 enkripcija, nikad plaintext AES-256
🔑 Lozinke Argon2id hash (najjači dostupni algoritam, 2025 standard) Argon2id
📋 Audit log Svaka akcija beleži se — append-only, bez brisanja Immutable log
🛑 Anti-hallucination Confidence threshold filter — AI ne daje odgovore ispod 62% pouzdanosti RAG guard
⚡ Rate limiting Max 60 zahteva/min po korisniku — zaštita od zloupotrebe Active
Tehničke specifikacije

🏗️ Infrastruktura svetske klase

⚡

FastAPI

Python backend. Asinhroni, brz, 40+ API endpointa. Auto-dokumentacija.

🧠

GPT-4o

OpenAI najjači model za pravne odgovore. GPT-4o-mini za klasifikaciju i glasovne komande.

🔍

Pinecone

Vektorska baza zakona. Semantička pretraga. Milioni vektora, milisekunde latencije.

🗄️

Supabase

PostgreSQL + Auth + RLS. Real-time sync. 16 migracija, 20+ tabela, indeksi, trigeri.

🚀

Railway

Cloudni deploy. Auto-deploy na git push. Dockerfile build, health check, auto-restart.

📱

PWA

Progressive Web App. Radi na telefonu kao aplikacija. Mobile bottom nav, touch 44px.

🎤

Web Speech API

Browser-native glasovno prepoznavanje. Srpski jezik (sr-RS). Nula eksternih zavisnosti.

📧

SEF + SMS

Sistem e-faktura integracija. SMS gateway za obaveštenja klijentima. GDPR compliant.
